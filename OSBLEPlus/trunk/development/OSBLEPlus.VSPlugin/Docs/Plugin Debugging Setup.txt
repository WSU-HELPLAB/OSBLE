To get debugging working,

1. set OSBIDE.Plugins.VS2013 to startup project
2. OSBIDE.Plugins.VS2013 properties select Start external program: and point it to your VS install devenv.exe, 
e.g. "C:\Program Files (x86)\Microsoft Visual Studio 12.0\Common7\IDE\devenv.exe"
3. under Startup Options / Command line arguments: add "/RootSuffix Exp"
